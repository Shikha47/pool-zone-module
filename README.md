# Pool Zone UI – Bootstrap 5.3.8

This project contains responsive UI screens built using Bootstrap 5.3.8 and SCSS.

Tech Stack

- HTML5
- Bootstrap 5.3.8
- SCSS (Sass)
- Node.js
- Live Server

Make sure Node.js is installed.
